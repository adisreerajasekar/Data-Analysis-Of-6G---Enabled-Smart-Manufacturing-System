# Data Analysis of 6G-Enabled Smart Manufacturing System

## Overview
This project focuses on analyzing data related to a 6G-enabled smart manufacturing system. The project explores how advanced connectivity, automation, and data-driven insights can improve manufacturing efficiency, monitoring, and decision-making.

The implementation includes data preprocessing, exploratory data analysis, visualization, and interpretation of important patterns within the smart manufacturing dataset.

---

## Problem Statement
Smart manufacturing systems generate large amounts of data from connected machines, sensors, and production environments. With 6G-enabled connectivity, these systems can support faster communication, low latency, automation, and intelligent decision-making.

This project aims to analyze manufacturing-related data to identify meaningful trends, performance patterns, and useful insights.

---

## Objectives
- Analyze smart manufacturing system data
- Perform data cleaning and preprocessing
- Explore key variables and relationships
- Create visualizations for better understanding
- Identify trends and patterns in the dataset
- Support data-driven decision-making in manufacturing
